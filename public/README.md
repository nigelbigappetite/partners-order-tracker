# Public Assets

Place your Hungry Tum logo file here.

## Logo Requirements

- **File name**: `hungry-tum-logo.png` or `hungry-tum-logo.svg`
- **Recommended size**: 40x40px to 80x80px
- **Format**: PNG (with transparency) or SVG

The logo will appear in the navigation bar at the top of every page.

If you have a different logo file name, update the `src` attribute in `components/Navigation.tsx`.
